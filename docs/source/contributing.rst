Contributing
------------

Thank you for taking your the time for contributing. We :icon:`fa fa-heart` your input or fix of any kind, whether it's:

* Reporting bugs
* Discussing the current state of the code
* Submitting a code fix
* Proposing new features
* Talking about us among your friends or on your social networks


:icon:`fa fa-seeding` Where the code grows
==========================================

We use github to host code and documentation, to track issues and feature requests, as well as accept pull requests.
